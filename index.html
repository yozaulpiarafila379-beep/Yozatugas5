1. Sealed Class

Sealed class adalah jenis class dalam Dart yang digunakan untuk membatasi pewarisan (inheritance), sehingga class tersebut hanya dapat diturunkan (di-extend) oleh class lain yang berada dalam file yang sama. Dengan adanya batasan ini, pengembang dapat mengontrol dengan jelas class mana saja yang menjadi turunan, sehingga struktur program menjadi lebih aman, terorganisir, dan mudah dipahami. Konsep sealed class juga membantu dalam memastikan bahwa semua kemungkinan turunan sudah diketahui, sehingga memudahkan saat melakukan pengecekan tipe (type checking) dalam program. Hal ini sangat berguna dalam pembuatan aplikasi yang membutuhkan alur logika yang jelas, seperti penanganan kondisi sukses dan error.

sealed class ApiResponse {}

class Success extends ApiResponse {
  final String message;
  Success(this.message);
}

class Error extends ApiResponse {
  final String errorMessage;
  Error(this.errorMessage);
}

void main() {
  print("Contoh penggunaan Sealed Class");
}

2. Keuntungan Sealed class

Sealed class adalah class dalam Dart yang membatasi pewarisan hanya dalam satu file, sehingga semua subclass harus dibuat dalam file yang sama. Hal ini membuat program lebih terkontrol dan mudah dipahami. Pada kode di atas, ApiResponse digunakan sebagai sealed class dengan dua turunan, yaitu Success dan Error, yang mewakili kondisi berhasil dan gagal. Melalui fungsi handleResponse(), program dapat melakukan pengecekan tipe untuk menampilkan hasil yang sesuai, sehingga alur program menjadi lebih rapi dan terstruktur.

Kode Program VS Code:


// Nama  : Yoza Ulfia Rafila
// NIM   : 251410004
// Kelas : SI2A

sealed class ApiResponse {}

class Success extends ApiResponse {
  final String message;
  Success(this.message);
}

class Error extends ApiResponse {
  final String errorMessage;
  Error(this.errorMessage);
}

void handleResponse(ApiResponse response) {
  if (response is Success) {
    print("Success: ${response.message}");
  } else if (response is Error) {
    print("Error: ${response.errorMessage}");
  }
}

void main() {
  handleResponse(Success("Data berhasil diambil"));
  handleResponse(Error("Terjadi kesalahan"));
}

3. Base Class

Polimorfisme adalah konsep dalam pemrograman berorientasi objek yang memungkinkan sebuah method memiliki banyak bentuk perilaku. Salah satu bentuk polimorfisme adalah method overriding, yaitu ketika subclass mendefinisikan ulang method yang sudah ada pada superclass. Pada kode di atas, class Animal memiliki method speak() yang menampilkan suara umum. Kemudian class Dog sebagai turunan dari Animal melakukan overriding pada method tersebut untuk menghasilkan suara yang berbeda, yaitu "Woof!". Hal ini menunjukkan bahwa method yang sama dapat memiliki perilaku yang berbeda tergantung pada objek yang memanggilnya.

Kode Program VS Code:


    // Nama  : Yoza Ulfia Rafila
// NIM   : 251410004
// Kelas : SI2A

class Animal {
  void speak() {
    print("Hewan bersuara");
  }
}

class Dog extends Animal {
  @override
  void speak() {
    print("Anjing: Woof!");
  }
}

void main() {
  Animal a = Animal();
  a.speak();

  Dog d = Dog();
  d.speak();
}
