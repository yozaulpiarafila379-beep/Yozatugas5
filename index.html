1.Sealed Class

  • Pada program ini, ApiResponse digunakan sebagai class utama yang memiliki dua turunan, yaitu Success dan Error.
  • Class Success digunakan untuk menyimpan pesan ketika proses berhasil.
  • Class Error digunakan untuk menyimpan pesan ketika terjadi kesalahan.
  • Kedua class tersebut merupakan turunan dari ApiResponse dan dibuat dalam satu file yang sama.
  • Dengan struktur ini, program hanya menggunakan dua kondisi, yaitu berhasil dan gagal.
  • Pada fungsi main(), program hanya menampilkan contoh penggunaan sealed class.

  Error(this.errorMessage);

  print("Contoh penggunaan Sealed Class");

2.Keuntungan Sealed Class

  • Pada program ini, ApiResponse digunakan sebagai class utama dengan dua turunan, yaitu Success dan Error.
  • Class Success digunakan saat data berhasil diambil, sedangkan Error digunakan saat terjadi kesalahan.
  • Pada fungsi handleResponse(), program mengecek tipe data menggunakan kondisi if (response is Success) dan if (response is Error).
  • Jika kondisi Success, maka program menampilkan pesan berhasil, sedangkan jika Error, program menampilkan pesan kesalahan.
  • Pada fungsi main(), program memanggil kedua kondisi tersebut untuk menampilkan hasil yang berbeda.

Kode Program VS Code:


// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI2A

sealed class ApiResponse {}

class Success extends ApiResponse {
  final String message;
  Success(this.message);
}

class Error extends ApiResponse {
  final String errorMessage;
  Error(this.errorMessage);
}

void handleResponse(ApiResponse response) {
  if (response is Success) {
    print("Success: ${response.message}");
  } else if (response is Error) {
    print("Error: ${response.errorMessage}");
  }
}

void main() {
  handleResponse(Success("Data berhasil diambil"));
  handleResponse(Error("Terjadi kesalahan"));
}

3.Polimorfisme (Method Overriding)

  • Pada program ini, class Animal memiliki method speak() yang menampilkan suara umum.
  • Class Dog merupakan turunan dari Animal yang melakukan overriding pada method speak().
  • Pada class Dog, method speak() diubah menjadi menampilkan "Anjing: Woof!".
  • Pada fungsi main(), objek Animal memanggil method speak() dengan output suara umum.
  • Objek Dog memanggil method speak() dengan output yang berbeda sesuai class turunan.
  • Hal ini menunjukkan bahwa method yang sama dapat menghasilkan output yang berbeda tergantung objek yang digunakan.

Kode Program VS Code:


// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI2A

class Animal {
  void speak() {
    print("Hewan bersuara");
  }
}

class Dog extends Animal {
  @override
  void speak() {
    print("Anjing: Woof!");
  }
}

void main() {
  Animal a = Animal();
  a.speak();

  Dog d = Dog();
  d.speak();
}

4.Method Overriding

  • Pada program ini, class Animal memiliki method speak() yang menampilkan suara umum.
  • Class Dog dan Cat merupakan turunan dari Animal yang melakukan overriding pada method speak().
  • Pada class Dog, method speak() menampilkan "Dog: Woof!", sedangkan pada class Cat menampilkan "Cat: Meow!".
  • Pada fungsi main(), objek Animal, Dog, dan Cat dipanggil menggunakan method speak().

5.Final Class

  • Pada program ini, Settings dibuat sebagai final class sehingga tidak dapat diwariskan ke class lain.
  • Class Settings memiliki atribut theme dan fontSize.
  • Method showSettings() digunakan untuk menampilkan nilai dari atribut tersebut.
  • Objek Settings dibuat pada fungsi main() dengan nilai awal tertentu.
  • Program kemudian memanggil method showSettings() untuk menampilkan hasil.
  • Penggunaan final class memastikan class tidak bisa diubah melalui pewarisan.

Kode Program VS Code:


// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI2A

final class Settings {
  String theme;
  int fontSize;

  Settings({required this.theme, required this.fontSize});

  void showSettings() {
    print("Theme: $theme, Font Size: $fontSize");
  }
}

void main() {
  Settings settings = Settings(theme: "Dark", fontSize: 14);
  settings.showSettings();
}

6.Inheritance

( Bonus 1 )
  • Pada program ini, Repository berperan sebagai class induk yang memiliki method create(), read(), update(), dan delete().
  • Class UserRepository merupakan turunan dari Repository dengan menggunakan keyword extends.
  • Class UserRepository mewarisi seluruh method dari class induk.
  • Method read() di-override untuk menampilkan data yang lebih spesifik yaitu data user.
  • Pada fungsi main(), objek UserRepository dibuat dan digunakan untuk memanggil semua method yang tersedia.
  • Program menampilkan hasil dari method create(), read(), update(), dan delete().

Kode Program VS Code:

// Kelas : SI2A

class Repository {

  }

    print("Menampilkan data");
  }

  void update() {
    print("Data berhasil diupdate");
  }

    print("Data berhasil dihapus");

8.Abstraksi

  • Pada program ini, Bentuk digunakan sebagai abstract class yang memiliki method hitungLuas().
  • Class Bentuk tidak dapat dibuat objek secara langsung.
  • Class Persegi, Lingkaran, dan Segitiga merupakan turunan dari Bentuk yang mengimplementasikan method hitungLuas().
  • Setiap class turunan menghitung luas dengan cara yang berbeda sesuai bentuknya.
  • Pada fungsi main(), objek dari masing-masing class dibuat untuk menghitung luas.
  • Program kemudian menampilkan hasil perhitungan luas persegi, lingkaran, dan segitiga.

Kode Dart:


// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI2A

// Class Abstrak
abstract class Bentuk {
  double hitungLuas();
}

// Class Persegi
class Persegi extends Bentuk {
  double sisi;

  Persegi(this.sisi);

  @override
  double hitungLuas() {
    return sisi * sisi;
  }
}

// Class Lingkaran
class Lingkaran extends Bentuk {
  double jariJari;

  Lingkaran(this.jariJari);

  @override
  double hitungLuas() {
    return 3.14 * jariJari * jariJari;
  }
}

// Class Segitiga
class Segitiga extends Bentuk {
  double alas;
  double tinggi;

  Segitiga(this.alas, this.tinggi);

  @override
  double hitungLuas() {
    return 0.5 * alas * tinggi;
  }
}

// Program Utama
void main() {
  // Objek Persegi
  Persegi persegi = Persegi(4);
  print("Luas Persegi: ${persegi.hitungLuas()}");

  // Objek Lingkaran
  Lingkaran lingkaran = Lingkaran(7);
  print("Luas Lingkaran: ${lingkaran.hitungLuas()}");

    
  // Objek Segitiga
  Segitiga segitiga = Segitiga(6, 8);
  print("Luas Segitiga: ${segitiga.hitungLuas()}");
}

10.Mixin

  • Pada program ini, terdapat dua mixin yaitu CanFly dan CanSwim.
  • Mixin CanFly memiliki state isFlying serta method fly() dan land().
  • Mixin CanSwim memiliki state isSwimming serta method swim() dan stopSwim().
  • Class Bird, Duck, Fish, dan Airplane menggunakan mixin dengan keyword with.
  • Class Duck menggunakan dua mixin sehingga memiliki kemampuan terbang dan berenang.
  • Pada fungsi main(), setiap objek dibuat dan method dipanggil sesuai kemampuan masing-masing.
  • Program kemudian menampilkan aktivitas seperti terbang, mendarat, dan berenang.

Kode Dart:


// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI2A

// Mixin dengan properti (state)
mixin CanFly {
  bool _isFlying = false;

  bool get isFlying => _isFlying;

  void fly() {
    if (!_isFlying) {
      _isFlying = true;
      print("Started flying!");
    } else {
      print("Already flying!");
    }
  }

  void land() {
    if (_isFlying) {
      _isFlying = false;
      print("Landed!");
    } else {
      print("Already on the ground!");
    }
  }
}

mixin CanSwim {
  bool _isSwimming = false;

  bool get isSwimming => _isSwimming;

  void swim() {
    if (!_isSwimming) {
      _isSwimming = true;
      print("Started swimming!");
    } else {
      print("Already swimming!");
    }
  }

  void stopSwim() {
    if (_isSwimming) {
      _isSwimming = false;
      print("Stopped swimming!");
    } else {
      print("Already stopped swimming!");
    }
  }
}

// Class dengan mixin (yang benar secara konsep OOP)
class Bird with CanFly {}

class Duck with CanFly, CanSwim {}

class Fish with CanSwim {}

class Airplane with CanFly {}

void main() {
  print("=== Bird ===");
  Bird bird = Bird();
  bird.fly();
  bird.land();

  print("\n=== Duck ===");
  Duck duck = Duck();
  duck.fly();
  duck.swim();
  duck.stopSwim();

  print("\n=== Fish ===");
  Fish fish = Fish();
  fish.swim();

  print("\n=== Airplane ===");
  Airplane airplane = Airplane();
  airplane.fly();
}

11.Sealed Class & Pattern Matching

  • Pada program ini, Status digunakan sebagai sealed class.
  • Terdapat tiga turunan yaitu Loading, Loaded, dan Failed.
  • Class Loaded menyimpan data, sedangkan Failed menyimpan pesan error.
  • Pada fungsi handleStatus(), digunakan switch-case untuk mengecek tipe objek.
  • Setiap kondisi diproses berbeda, yaitu menampilkan loading, data, atau error.
  • Pada fungsi main(), program memanggil handleStatus() dengan kondisi yang berbeda.
  • Program kemudian menampilkan hasil sesuai dengan status yang diberikan.

Kode Dart:


// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI2A

// Sealed class
sealed class Status {
  const Status();
}

class Loading extends Status {
  const Loading();
}

class Loaded extends Status {
  final String data;
  const Loaded(this.data);
}

class Failed extends Status {
  final String error;
  const Failed(this.error);
}

// Function handle status
void handleStatus(Status status) {
  switch (status) {
    case Loading():
      print("Loading...");
      return;

    case Loaded(:final data):
      print("Data: $data");
      return;

    case Failed(:final error):
      print("Error: $error");
      return;
  }
}

void main() {
  handleStatus(const Loading());
  handleStatus(const Loaded("Data berhasil"));
  handleStatus(const Failed("Terjadi error"));
}

12.State Management dengan Sealed Class

  • Pada program ini, AppState digunakan sebagai sealed class.
  • Terdapat tiga turunan yaitu LoadingState, SuccessState, dan ErrorState.
  • Class SuccessState menyimpan data berupa list item, sedangkan ErrorState menyimpan pesan error.
  • Pada fungsi updateUI(), digunakan switch-case untuk mengecek tipe state.
  • Setiap state akan memanggil fungsi yang berbeda yaitu showLoading(), showItems(), dan showError().
  • Pada fungsi main(), program memanggil updateUI() dengan kondisi loading, success, dan error.
  • Program kemudian menampilkan output sesuai dengan state yang diberikan.

Kode Dart:


// Nama  : Yoza Ulpia Rafila
// NIM   : 251410004
// Kelas : SI2A

// Sealed class
sealed class AppState {
  const AppState();
}

// State Loading
class LoadingState extends AppState {
  const LoadingState();
}

// State Success
class SuccessState extends AppState {
  final List items;

  const SuccessState(this.items);
}

// State Error
class ErrorState extends AppState {
  final String message;

  const ErrorState(this.message);
}

// Handler (Pattern Matching)
void updateUI(AppState state) {
  switch (state) {
    case LoadingState():
      showLoading();
      break;

    case SuccessState(:final items):
      showItems(items);
      break;

    case ErrorState(:final message):
      showError(message);
      break;
  }
}

// UI Functions
void showLoading() {
  print("⏳ Loading...");
}

void showItems(List items) {
  print("✅ Success! Items:");
  for (int i = 0; i < items.length; i++) {
    print("${i + 1}. ${items[i]}");
  }
}

void showError(String message) {
  print("❌ Error: $message");
}

// Main Program
void main() {
  updateUI(const LoadingState());

  updateUI(const SuccessState([
    "Item A",
    "Item B",
    "Item C"
  ]));

  updateUI(const ErrorState("Gagal memuat data"));
}

13.Kesimpulan

Dari seluruh pembahasan yang telah dilakukan, dapat disimpulkan bahwa:

  • Sealed class membatasi pewarisan dalam satu file sehingga lebih terkontrol.
  • Final class mencegah pewarisan agar struktur tetap konsisten.
  • Abstract class berfungsi sebagai dasar bagi class turunan.
  • Mixin digunakan untuk menambahkan perilaku tanpa inheritance.
  • Inheritance memungkinkan penggunaan kembali kode.
  • Polimorfisme memungkinkan method memiliki perilaku yang berbeda.
  • Seluruh konsep tersebut membuat program lebih terstruktur, fleksibel, dan mudah dikembangkan.
